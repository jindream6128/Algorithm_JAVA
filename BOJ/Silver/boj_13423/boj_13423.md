# 13423: Three Dots (Silver2)
[주소](https://www.acmicpc.net/problem/13423)

## solve
점 N 개가 주어지고 그중에서 3개를 골라서 가운데 점을 기준으로 같은거리에 떨어져있는 점의 쌍을 찾기
-4 -1 0 2 4
3가지

첫번째줄 테스트 케이스 t
t개의 테스트 케이스
점은 +-10억 -> int

출력은 가능한 경우의수 cnt출력

---
2 0 -4 -1 4

-4 -1 0 2 4

이진탐색인가?
정렬을하고 처음 s 마지막꺼 e = s+2
해서 s+e /2 의값이 s+1 이면cnt++
e 는 n-1 까지
s 는 n-3까지
