# 3733: Shares (Bronze5)

## solve

입력 텍스트 파일에서 정수 숫자 쌍을 읽는 프로그램을 작성하십시오. 각 쌍은 순서대로 1 ≤ N ≤ 10000 및 1 ≤ S ≤ 109의 값을 포함합니다. 입력 데이터는 자유롭게 공백으로 구분되며 올바르며 파일의 끝으로 종료됩니다. 각 숫자 쌍에 대해 프로그램은 x의 최대값을 계산하고 그 값을 아래와 같이 표준 출력에 출력합니다.
